# bigdata_rest

通过 rest 访问 impala 和 hbase
好处：
1.方便各种语言、系统调用
2.可以对查询的语句进行权限管理

工具springboot

实现原理：
1.用户通过 rest 访问 本系统，系统再通过调用与impala的连接池访问 impala。
2.用户通过 rest 访问 本系统，系统通过把 用户的请求进行转发，调用hbase的rest 服务。
